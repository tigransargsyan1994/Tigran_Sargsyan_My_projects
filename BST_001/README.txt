This project makes Binary Search Tree from given numbers.
'''Example of tree'''

         15
      /      \
    7          22

'Preorder' button type of output shows whole tree in top-left-right formation (15-7-22).
'Inorder' button type of output shows whole tree in left-top-right formation  (7-15-22).
'Preorder' button type of output shows whole tree in left-right-top formation (7-22-15).
'Count' button shows how many elements are in the tree.
'Height' button shows hom many levels are in the tree.
'Clear space' button removes all given numbers.
